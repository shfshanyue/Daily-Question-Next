# 实现 LazyMan

> 更多描述

```js
LazyMan("Hank");
// 输出：
// Hi!This is Hank!

LazyMan("Hank").sleep(10).eat("dinner");
// 输出：
// Hi! This is Hank!
// 等待10秒..
// Wake up after 10
// Eat dinner~

LazyMan("Hank").eat("dinner").eat("supper");
// 输出：
// Hi This is Hank!
// Eat dinner~
// Eat supper~

LazyMan("Hank").sleepFirst(5).eat("supper");
// 输出：
// 等待5秒..
// Wake up after 5
// Hi This is Hank!
// Eat supper
```
