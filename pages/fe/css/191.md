> Author
> 回答者: [cy948](https://github.com/cy948)

```css
.triangle {
  margin: 0 auto;
  width: 50vh;
  height: 50vh;
  background-color: aqua;
  /* 
                画多边形： 
                polygon(x y, x y, ...)
                一组 x y, 代表一个点
                x: 往右增加
                y: 往下增加
            */
  clip-path: polygon(0 0, 0% 100%, 100% 100%);
}
```
