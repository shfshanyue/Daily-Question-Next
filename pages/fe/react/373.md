# 关于 React hooks 的 caputre value，以下输出多少

> 更多描述

```js
function App() {
  const [count, setCount] = useState(0);
  const incr = () => {
    setTimeout(() => {
      setCount(count + 1);
    }, 3000);
  };
  return <h1 onClick={incr}>{count}</h1>;
}
```

当连续点击 10 次时，会输出多少
