# 什么是公有云，私有云，混合云以及多重云

> Issue
> 欢迎在 Gtihub Issue 中回答此问题: [Issue 166](https://github.com/shfshanyue/Daily-Question/issues/166)

> Author
> 回答者: [timtike](https://github.com/timtike)

公有云就是阿里云 腾讯云 aws 等
私有云 就是公司自己买物理机，在机房自己搭建网络，自己做虚拟机
混合云 就是 公有云 + 私有云
多重云 就是多个公有云
