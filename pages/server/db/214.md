# 什么是隔离级，都有哪些隔离级

> Issue
> 欢迎在 Gtihub Issue 中回答此问题: [Issue 214](https://github.com/shfshanyue/Daily-Question/issues/214)

> Author
> 回答者: [shfshanyue](https://github.com/shfshanyue)

隔离级事关并发事务的隔离机制，ANSI SQL 中定义了四种隔离级，分别是

- Read Uncommited
- Read Committed
- Repetable Read
- Serializable
