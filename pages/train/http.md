## 其它

### 需要什么基础？

+ 该专栏不适合毫无编程经验的同学
+ 会涉及到简单的 HTTP Server，因此需要有一点 Node.js 基础，不过可现学。

### 达成什么目标？

能够回答大部分前端面试中关于 HTTP 的部分，加强前后端沟通桥梁 API 的理解，并加深对 `axios`/`fetch` 等理解，能手写简单的爬虫以及 HTTP 客户端，比如 `axios` 类。
